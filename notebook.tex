    \section{Graphlet Correlation
Analysis}\label{graphlet-correlation-analysis}

Remember, in notebook \emph{Thresholding}, I thresholded contact maps
and saved so that orca can read them.

In this notebook, I perform two actions: 1. I apply \texttt{count5}
procedure in orca to extract 73 orbits for each loci in each chromosome.
After this stage, graphlets will be save with \texttt{.graphlets}
extension in the \texttt{./data} folder.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\setcounter{enumi}{1}
\tightlist
\item
  I then read the graphlets and find both loci-wise distance and
  graphlet-wise correlations between the four cell lines.
\end{enumerate}

By repeating steps 1 and 2 for all 23 chromosomes, I will have pairwise
MIC values for all orbits of all cell lines. I would eventually be able
to have a data array of shape \texttt{A(23\ *\ 73)}, for each pair of
cell lines where \texttt{A{[}i,\ j{]}} denotes the correlation between
the jth orbit in the ith chromosome.
